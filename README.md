# Time Tracker

컴퓨터 사용 시간과 프로그램별 사용 시간을 자동으로 기록해주는 Windows 전용 프로그램입니다.

## 기능

- 전체 컴퓨터 사용 시간 기록  
- **프로그램별 사용 시간** 자동 측정 (예: chrome.exe, code.exe 등)  
- 사용자가 **마우스/키보드를 5초 이상 사용하지 않으면 자동 일시정지**  
- Visual Studio Code가 **꺼져 있을 경우에는 시간 강제 기록 유지**
- GUI를 통한 간편한 조작 (시작 / 멈춤 / 초기화 버튼)

### 1. 파이썬으로 실행
